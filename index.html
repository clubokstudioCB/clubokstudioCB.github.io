Privacy Policy

Requested permissions:

  • Camera allows for scanning QR code containing an API key. Immediately after parsing the QR code, the image is deleted from memory.
  • Full network access enables the app to connect to the Binance.

Security:

The API key is stored only on the device and used to connect to the Binance account via the API.

Do not install hacking programs. API key will open access to all assets on your Binance account.
